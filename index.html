Engineering Projects

Facial Recognition Security System

Project Description
Utilized C# and created a Visual Studio IoT project that took the data from a Microsoft Webcam and determined whether a face was detected in the photo taken. If a face was detected, it would then take that photo and compare it to the list of authorized users, using Microsoft Cognitive Services: Face API to process the authentification. Depending on whether the person in front of the camera is an authorized user or not, the IoT device (Dragonboard 410C) would output a voltage to the Electric strike connected to the door, enabling the door to become unlocked and opened. If an unauthorized user were to intent on opening the door, no action would be taken, as only an authorized user can unlock the door.
Equipment Used
  • Dragonboard 410C IoT Board
  • Visual Studio 2015
  • C#
  • Windows 10 IoT Core
  • 12V Fail-Safe Electric Strike
  • 5V Relay
TimeFrame
This project was worked on in collaboration for over a span of approximately twelve months.
